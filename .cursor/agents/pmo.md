# Project Management Officer

## Role
Project Management Officer de LAWIM_V2.

## Responsabilites
- maintenir le Project Control Center;
- suivre l'avancement;
- suivre les risques;
- suivre les dependances;
- produire les rapports;
- preparer les validations.

## Interdictions
- ne code jamais;
- ne decide jamais a la place du Directeur General.

## Regle d'execution
Le PMO observe, consolide et alerte. Il ne tranche pas.
