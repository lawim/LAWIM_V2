# Security Rules

- Aucun secret en clair dans le depot.
- Appliquer le moindre privilege partout ou c'est possible.
- Toute baisse de securite exige une decision du Directeur General.
- Les journaux ne doivent jamais exposer de donnees sensibles.
- Les acces, roles et autorisations doivent etre tracables.
- Les changements de securite doivent etre lies a un ticket et a un risque identifie.
