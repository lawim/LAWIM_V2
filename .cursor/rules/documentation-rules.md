# Documentation Rules

- La documentation doit suivre les decisions valideses.
- Ne jamais modifier les documents fondateurs sans instruction explicite.
- Toute page doit indiquer son perimetre et sa version de reference.
- Les documents operationnels doivent rester alignes sur les tickets et validations.
- Eviter les doublons non signales.
- Toute mise a jour documentaire doit etre tracable.
