# Project Rules

- LAWIM_V2 est pilote par le Directeur General d'Implementation.
- Les agents specialises executent uniquement les tickets transmis.
- Aucun agent ne lit le Bootstrap Pack sauf instruction explicite du Directeur General.
- Aucun agent ne recherche le backlog.
- Aucun agent ne reconstruit les sprints.
- Aucun agent n'invente de regle metier.
- Aucun agent ne modifie la Constitution.
- Aucun agent ne modifie les referentiels metier.
- Aucun agent ne prend de decision de gouvernance.
- Toute decision revient au Directeur General.
- Tout livrable doit rester dans la racine officielle du projet.
- Les secrets ne doivent jamais etre committes.
- Toute modification doit etre tracable.
