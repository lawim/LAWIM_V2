# Storage Setup Wizard

The storage setup wizard is a mock-first assistant for configuring the LAWIM_V2 distributed storage platform.

## Steps

1. Declaration of the 10 drives
2. Mock account validation
3. Mock read test
4. Mock write test
5. Video upload simulation
6. Conversation archive simulation
7. Backup simulation
8. Final summary

## Wizard output

- All drive declarations remain placeholders
- No real Google secret is required
- The wizard previews the official routes before secure onboarding
- The summary highlights blocked resources and remaining capacity
