# RELEASE_CLASSIFICATION

## Classification officielle

| Release | Nom officiel | Objectif |
|---|---|---|
| AAD | Microsoft Entra ID / Identity Provider / Authentication Scaffold | Intégration optionnelle d’authentification et d’identité |
| AAE | Production Security & Secrets | Sécurité de production, secrets, pare-feu, SSH, Nginx, chiffrement |
| AAF | Migration OVH | Migration de stockage et d’infrastructure |
| AAG | Monitoring & Observability | Monitoring, métriques, alertes, traçabilité |
| AAH | Performance & Load Testing | Performance, charge, stabilité |
| AAI | Go Live Certification | Certification de mise en production |
| AAJ | Production Deployment | Déploiement production |

## Décision officielle

AAD est désormais réservée à Microsoft Entra ID et à l’architecture d’Identity Provider optionnelle. Elle ne désigne plus la sécurité globale de production.
