# Deployment Checklist

## Infrastructure
- Confirm OS, kernel, and host requirements.
- Validate Docker, Docker Compose, Git, Node.js, and Python.

## Database
- Confirm PostgreSQL reachability and backup workflow readiness.

## Frontend
- Verify build pipeline, asset output, and routing readiness.

## Backend
- Validate health endpoints and operational readiness.

## Brain / Knowledge / Agents
- Verify service health endpoints and dependency connectivity.

## Monitoring / Backup / Restore / Security
- Confirm monitoring endpoints, backup validation, and security checks.
