# Google Drive Setup

The setup wizard prepares LAWIM_V2 for real Google Drive onboarding without modifying the code later.

## Steps

1. Declare the 10 Google Drive resources
2. Validate OAuth placeholders
3. Run the connection test
4. Run the read test
5. Run the write test
6. Create the automatic folders
7. Simulate video upload and conversation archive
8. Simulate backup activation and verification

## Automatic folders

- VIDEOS
- VIDEOS_ARCHIVE
- PHOTOS
- AUDIO
- DOCUMENTS
- CONVERSATIONS
- BACKUPS
- EXPORTS
- TEMP
- LOGS

## Wizard guarantees

- Placeholders only
- No real Google secret is requested
- Official routes are previewed before activation
- The summary surfaces capacity, alerts, and OAuth readiness
