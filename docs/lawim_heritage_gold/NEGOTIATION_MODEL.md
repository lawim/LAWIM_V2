## 3. Closing Techniques

**Source :** LAWIM `KNOWLEDGE/commercial/closing_techniques.md`

### 3.1 5-Step Trust Sequence

| Step | Nom | Action |
|------|-----|--------|
| 1 | **Écoute active** | Reformuler les besoins du client, montrer qu'on comprend |
| 2 | **Apport d'information** | Fournir des données précises (prix, docs, quartier) |
| 3 | **Proposition** | Présenter une solution adaptée au besoin exprimé |
| 4 | **Traitement des objections** | Répondre point par point aux résistances |
| 5 | **Closing** | Proposer le passage à l'action (visite, offre, contact) |

### 3.2 Visit-Based Closing

- **Principe :** La visite est le moment de closing le plus efficace
- **Processus :**
  1. Proposer la visite dès que l'intérêt est confirmé
  2. Préparer les documents avant la visite
  3. Pendant la visite : valoriser les points forts, répondre aux objections en direct
  4. Après la visite : relance à J+1 pour capitaliser sur l'émotion

### 3.3 Document-Based Closing

- **Principe :** Les documents juridiques rassurent et finalisent
- **Processus :**
  1. Partager le titre foncier ou la promesse de vente
  2. Expliquer chaque document simplement
  3. Proposer l'accompagnement notaire LAWIM (50k FCFA)
  4. Closing sur la sécurité juridique

### 3.4 No Aggressive Closing on "Dernier Prix"

- **Règle :** Quand le client demande le "dernier prix", ne pas faire de forcing
- **Réponse :** Donner le prix en contexte, expliquer la marge, proposer une discussion
- **Interdit :** "C'est le dernier prix, prenez-le ou laissez-le"

### 3.5 8-Step Commercial Process

| Step | Nom | Description |
|------|-----|-------------|
| 1 | **Prospection** | Identifier et attirer des leads |
| 2 | **Qualification** | Comprendre le besoin, le budget, la localisation |
| 3 | **Présentation** | Montrer les biens correspondants |
| 4 | **Proposition** | Faire une offre adaptée |
| 5 | **Suivi** | Relancer, répondre aux questions |
| 6 | **Conclusion** | Finaliser la transaction |
| 7 | **Activation** | Mettre en service (emménagement, mise en location) |
| 8 | **Fidélisation** | Garder le contact pour futures transactions |

## 6. Price Negotiation

**Source :** LAWIM `KNOWLEDGE/negotiation-patterns.md`

### 6.1 Six Price Levels

| Niveau | Définition | Usage |
|--------|-----------|-------|
| **Prix affiché** | Prix publié sur l'annonce | Point de départ visible |
| **Prix négociable** | Marge réelle de négociation | À déterminer avec le vendeur |
| **Prix final** | Prix de transaction effectif | Résultat de la négociation |
| **Prix estimation** | Prix du marché estimé | Pour conseiller le vendeur |
| **Fourchette de prix** | Intervalle haut-bas | Pour cadrer la recherche |
| **Prix historique** | Prix précédents du bien | Pour justifier l'évolution |

### 6.2 Price Normalization

| Dimension | Application |
|-----------|-------------|
| **Devise** | Convertir systématiquement en FCFA |
| **Période** | Préciser la date de l'estimation |
| **Source** | Indiquer la provenance du prix (annonce, estimation, transaction) |
| **Contexte** | Expliquer le contexte du prix (marché, quartier, état du bien) |
| **Niveau de négociation** | Préciser si le prix est ferme, négociable, ou à débattre |
| **Historique** | Mentionner les variations de prix dans le temps |

### 6.3 Cultural Dynamics of Cameroon Negotiation

| Règle | Explication |
|-------|-------------|
| **Le vendeur fixe un prix ambitieux** | Le prix affiché inclut une marge de négociation |
| **Le vendeur attend la première offre** | Le vendeur ne donne pas son prix plancher |
| **Remise pour transaction rapide** | Paiement comptant ou rapide = levier de baisse |
| **Offre basse = ouverture** | Une offre basse est une porte d'entrée, pas une insulte |
| **L'acheteur négocie tôt** | Les Camerounais négocient dès le premier contact |
| **Contre-offre = test** | La contre-offre du vendeur est souvent encore négociable |
| **Preuve comme levier** | Montrer des biens similaires moins chers pour justifier une baisse |
