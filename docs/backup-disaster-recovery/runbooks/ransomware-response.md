# Runbook - Ransomware Response

STATUT : A COMPLETER

## Trigger

- destructive encryption or mass deletion is suspected

## First actions

1. Disconnect external media.
2. Stop synchronization and destructive jobs.
3. Isolate the server.
4. Preserve logs and evidence.
5. Restore from a clean older backup in a safe environment.

## Recovery source

- historical clean backup
- external disk if its date is safe
