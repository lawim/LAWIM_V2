# External Disk

STATUT : CIBLE DOCUMENTAIRE ACTIVE

## Role

The external disk is the physically separated offline copy used for disaster recovery when Google Drive is unavailable.

## Policy target

- Weekly copy
- Multiple disks in rotation
- Preferably two labeled media: Disk A and Disk B
- Each disk must be mounted, verified, copied, checked, and safely disconnected

## Physical handling

- Protect against humidity
- Protect against heat
- Protect against theft
- Keep a responsible operator record
- Log each rotation
