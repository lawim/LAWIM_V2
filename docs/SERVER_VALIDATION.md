# Server Validation

## Checklist

- Confirm OS and kernel compatibility.
- Validate Docker and Docker Compose installation.
- Confirm Node.js and Python runtime availability.
- Validate Redis and PostgreSQL reachability.
- Ensure ports, firewall, DNS, and internet access are available.
- Check storage, memory, and CPU sizing.
- Confirm timezone, locale, permissions, and TLS settings.
- Validate backup and restore simulation readiness.
