# Procédure de certification des agences

Version 1.0

---

# 1. Objectif

Le présent document définit la procédure de certification des agences LAWIM.

---

# 2. Périmètre

Cette procédure couvre :

* les conditions d'éligibilité ;
* les documents exigés ;
* la vérification de l'existence réelle ;
* la certification ;
* le renouvellement ;
* le retrait de certification ;
* l'affichage dans les outils internes.

---

# 3. Niveaux possibles

* agence enregistrée ;
* agence vérifiée ;
* agence certifiée LAWIM ;
* agence suspendue ;
* agence radiée.

---

# 4. Acteurs concernés

* partenariat ;
* conformité ;
* opérations ;
* sécurité ;
* direction.

---

# 5. Conditions d'éligibilité

L'agence doit présenter :

* une existence réelle ;
* un responsable identifié ;
* un portefeuille de biens ou de missions ;
* des moyens de contact stables ;
* une activité cohérente ;
* une volonté de respecter les règles LAWIM ;
* une compréhension du modèle sans commission immobilière.

---

# 6. Documents exigés

Demander au minimum :

* RCCM ou preuve d'existence légale équivalente ;
* NIF si disponible ;
* CNI ou passeport du responsable ;
* identité du responsable ;
* preuve d'existence légale ;
* adresse ou siège ;
* contacts ;
* liste d'équipe si utile ;
* références de biens ou missions ;
* exemples d'annonces ;
* références clients si disponibles ;
* coordonnées de paiement si nécessaires.

---

# 7. Critères de certification

La certification peut être accordée si :

* les documents sont cohérents ;
* l'activité réelle est démontrée ;
* les annonces sont de qualité acceptable ;
* le niveau de litige est maîtrisé ;
* les réponses sont rapides et sérieuses ;
* les règles LAWIM sont acceptées ;
* aucun risque majeur n'est détecté.

---

# 8. Durée de validité

La certification est valide 12 mois sauf suspension, retrait ou décision exceptionnelle.

Le renouvellement doit être revu avant échéance.

---

# 9. Retrait de certification

Retirer la certification si :

* fraude ;
* plaintes répétées ;
* documents devenus invalides ;
* comportement abusif ;
* non-respect des règles ;
* informations mensongères ;
* activité suspecte ;
* litiges non résolus.

---

# 10. Impact sur LAWIM

La certification influence :

* la visibilité dans les parcours internes ;
* la confiance perçue ;
* la priorisation des interactions ;
* le scoring de confiance ;
* le matching lorsque la logique métier le permet.

---

# 11. Notifications

Notifier :

* l'agence ;
* le responsable interne ;
* le support ;
* la direction si certification ou retrait sensible.

---

# 12. Reporting

Suivre au minimum :

* nombre d'agences enregistrées ;
* nombre d'agences vérifiées ;
* nombre d'agences certifiées ;
* nombre de retraits ;
* motifs de retrait ;
* délais moyens de certification ;
* taux de renouvellement.

---

# 13. Règles absolues

* ne jamais certifier sur la base d'une impression ;
* ne jamais promettre un avantage commercial garanti ;
* ne jamais créer de commission immobilière ;
* ne jamais contourner la conformité ;
* ne jamais ignorer un signal de fraude.

---

# 14. Objectif final

Cette procédure garantit que les agences présentes dans LAWIM sont identifiées, vérifiées et suivies dans la durée.
