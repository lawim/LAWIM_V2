# Procédure d'intégration des partenaires

Version 1.0

---

# 1. Objectif

Le présent document définit la procédure d'intégration des partenaires LAWIM.

Il s'applique aux agences, promoteurs, notaires, géomètres, artisans, photographes, vidéastes, prestataires de visite, prestataires administratifs et partenaires financiers éventuels.

---

# 2. Périmètre

Cette procédure couvre l'entrée d'un partenaire dans l'écosystème LAWIM, depuis le premier contact jusqu'à l'activation et au suivi périodique.

Elle ne crée aucune commission immobilière.

---

# 3. Acteurs concernés

* équipe partenariat ;
* support ;
* conformité ;
* administration ;
* sécurité ;
* équipe commerciale ;
* direction si validation exceptionnelle.

---

# 4. Étapes officielles

1. Identification du partenaire.
2. Premier contact.
3. Présentation de LAWIM.
4. Collecte des informations.
5. Vérification minimale.
6. Validation documentaire.
7. Création du compte partenaire.
8. Attribution du rôle.
9. Formation rapide.
10. Activation.
11. Suivi.
12. Évaluation périodique.

---

# 5. Documents à demander

Selon le type de partenaire, demander au minimum :

* nom légal ou nom commercial ;
* RCCM ou preuve d'existence équivalente ;
* NIF si disponible ;
* CNI ou passeport du responsable ;
* identité du responsable ;
* document d'immatriculation ou preuve d'existence ;
* attestation de localisation si disponible ;
* adresse ou localisation principale ;
* numéro de téléphone ;
* adresse e-mail ;
* portfolio ou références ;
* liste des services proposés ;
* preuve d'activité réelle ;
* coordonnées de paiement si nécessaire ;
* autorisations sectorielles si elles existent ;
* références de clients si disponibles.

---

# 6. Critères d'acceptation

Le partenaire peut être accepté si :

* l'identité est cohérente ;
* les documents sont lisibles et plausibles ;
* le contact fonctionne ;
* l'activité est réelle ;
* le service proposé est utile à LAWIM ;
* le partenaire accepte les règles LAWIM ;
* le partenaire comprend qu'il n'existe aucune commission immobilière.

---

# 7. Critères de refus

Refuser l'intégration si :

* les informations sont fausses ou contradictoires ;
* le contact est injoignable ;
* les documents sont absents ou manifestement faux ;
* le partenaire refuse les règles LAWIM ;
* le partenaire demande une commission sur vente ou location immobilière ;
* un risque de fraude est détecté.

---

# 8. Délais

Les délais internes recommandés sont :

* premier retour : 24 heures ouvrées ;
* vérification minimale : 48 heures ouvrées ;
* validation documentaire : 3 jours ouvrés ;
* activation : après validation complète.

---

# 9. Responsabilités

* le partenariat collecte et relance ;
* la conformité vérifie les documents ;
* la sécurité traite les signaux de risque ;
* l'administration crée ou met à jour le compte ;
* la direction arbitre les cas sensibles.

---

# 10. Traçabilité

Chaque dossier doit conserver :

* la date du premier contact ;
* le nom du responsable interne ;
* les documents reçus ;
* le résultat de vérification ;
* la décision ;
* la date d'activation ou de refus ;
* les motifs de refus éventuels ;
* l'historique des relances.

---

# 11. Notifications

Notifier au minimum :

* la réception du dossier ;
* la demande de pièces manquantes ;
* l'acceptation ;
* le refus ;
* l'activation ;
* la mise en attente ;
* la clôture du dossier.

---

# 12. Reporting

Suivre au minimum :

* nombre de partenaires contactés ;
* nombre de dossiers complets ;
* nombre de partenaires activés ;
* nombre de refus ;
* délai moyen d'intégration ;
* taux de conversion contact -> activation ;
* nombre de partenaires par catégorie.

---

# 13. Risques

* faux partenaire ;
* documents falsifiés ;
* prestataire non fiable ;
* partenaire non actif ;
* attente prolongée ;
* incompréhension du modèle sans commission.

---

# 14. Règles absolues

* ne jamais promettre un revenu garanti ;
* ne jamais promettre une vente garantie ;
* ne jamais promettre une location garantie ;
* ne jamais créer de commission immobilière ;
* toujours documenter les décisions ;
* toujours valider humainement les cas sensibles.

---

# 15. Objectif final

Cette procédure permet d'intégrer des partenaires utiles, vérifiés et traçables dans l'écosystème LAWIM.
