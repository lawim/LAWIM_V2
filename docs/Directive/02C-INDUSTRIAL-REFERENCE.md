# Référentiel spécialisé des biens industriels

Version 1.0

---

# CHAPITRE 1 — OBJECTIF

Le présent document définit les spécificités de la famille industrielle.

Il hérite du référentiel maître des biens immobiliers.

---

# CHAPITRE 2 — PÉRIMÈTRE

Les biens industriels couvrent notamment :

* entrepôt ;
* dépôt ;
* usine ;
* atelier ;
* plateforme logistique ;
* site de production ;
* local de stockage lourd ;
* espace technique.

---

# CHAPITRE 3 — ATTRIBUTS INDUSTRIELS SPÉCIFIQUES

Les attributs les plus utiles sont notamment :

* accès camion ;
* hauteur sous plafond ;
* charge au sol ;
* aire de manœuvre ;
* quai de chargement ;
* puissance électrique ;
* eau industrielle ;
* sécurité périmétrique ;
* zone de nuisance ;
* conformité de zonage ;
* surface utile ;
* réserve foncière.

---

# CHAPITRE 4 — RÈGLES DE PUBLICATION

Un bien industriel doit préciser :

* le type d'activité autorisée ;
* les contraintes techniques ;
* les accès logistiques ;
* le niveau d'équipement ;
* la compatibilité réglementaire.

---

# CHAPITRE 5 — RÈGLES DE MATCHING

Le matching industriel valorise :

* les accès poids lourds ;
* la capacité d'exploitation ;
* la sécurité ;
* la puissance disponible ;
* la disponibilité des services techniques ;
* la proximité des axes logistiques.

---

# CHAPITRE 6 — RÈGLES DE VALIDATION

Un bien industriel est invalide si :

* le site ne supporte pas l'usage annoncé ;
* les contraintes techniques sont absentes alors qu'elles sont indispensables ;
* la logistique est incompatible avec le site ;
* le prix ne correspond pas à la capacité du bien.

---

# CHAPITRE 7 — TESTS SPÉCIFIQUES

Le référentiel industriel doit couvrir au minimum :

* entrepôt ;
* dépôt ;
* usine ;
* atelier ;
* plateforme logistique.

---

# CHAPITRE 8 — OBJECTIF FINAL

Le référentiel industriel doit permettre à LAWIM de qualifier les biens à usage productif et logistique avec précision et prudence.

---

# FIN DU DOCUMENT

Le présent **02C-INDUSTRIAL-REFERENCE.md** constitue le référentiel spécialisé des biens industriels de LAWIM.
