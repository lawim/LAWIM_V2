# 02D-LAND-REFERENCE.md

# Référentiel spécialisé des biens fonciers

Version 1.0

---

# CHAPITRE 1 — OBJECTIF

Le présent document définit les spécificités de la famille foncière.

Il hérite du référentiel maître des biens immobiliers.

---

# CHAPITRE 2 — PÉRIMÈTRE

Les biens fonciers couvrent notamment :

* terrain nu ;
* parcelle ;
* lot ;
* terrain titré ;
* terrain non titré ;
* terrain à bâtir ;
* terrain viabilisé ;
* terrain agricole urbain ;
* terrain de réserve.

---

# CHAPITRE 3 — ATTRIBUTS FONCIERS SPÉCIFIQUES

Les attributs les plus utiles sont notamment :

* superficie ;
* limites / bornes ;
* titre foncier ;
* type de sol ;
* accessibilité ;
* topographie ;
* viabilisation ;
* servitudes ;
* vocation ;
* zonage ;
* documents de propriété ;
* largeur d'accès.

---

# CHAPITRE 4 — RÈGLES DE PUBLICATION

Un bien foncier doit préciser :

* son statut juridique ;
* sa superficie ;
* son accès ;
* son usage possible ;
* son niveau de viabilisation.

---

# CHAPITRE 5 — RÈGLES DE MATCHING

Le matching foncier valorise :

* la surface ;
* le titre ;
* l'accès ;
* la constructibilité ;
* la proximité des repères ;
* la vocation urbanistique ou économique.

---

# CHAPITRE 6 — RÈGLES DE VALIDATION

Un bien foncier est invalide si :

* sa vocation n'est pas compréhensible ;
* sa superficie n'est pas fiable ;
* son statut juridique est absent alors qu'il est requis ;
* un document critique est incohérent.

---

# CHAPITRE 7 — TESTS SPÉCIFIQUES

Le référentiel foncier doit couvrir au minimum :

* terrain nu ;
* terrain titré ;
* terrain à bâtir ;
* parcelle ;
* terrain viabilisé.

---

# CHAPITRE 8 — OBJECTIF FINAL

Le référentiel foncier doit permettre à LAWIM de traiter les terrains et parcelles avec une logique juridique, géographique et commerciale maîtrisée.

---

# FIN DU DOCUMENT

Le présent **02D-LAND-REFERENCE.md** constitue le référentiel spécialisé des biens fonciers de LAWIM.
