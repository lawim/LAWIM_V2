# CHAPITRE 1 — OBJECTIF

Le présent document définit le dictionnaire métier canonique de LAWIM.

Il constitue la base commune de :

* la conversation ;
* le matching ;
* la traduction ;
* la recherche tolérante ;
* LAWIM AI ;
* les tests multilingues.

---

# CHAPITRE 2 — STRUCTURE D'UN CONCEPT

Chaque concept doit pouvoir contenir au minimum :

* identifiant ;
* catégorie ;
* français ;
* anglais ;
* pidgin ;
* synonymes ;
* variantes ;
* expressions locales ;
* fautes fréquentes ;
* catégorie immobilière ;
* poids sémantique ;
* niveau de confiance.

---

# CHAPITRE 3 — PRINCIPES

Le dictionnaire doit :

* couvrir plusieurs centaines de concepts ;
* normaliser les formulations locales ;
* conserver les variantes utiles ;
* relier un concept à une ou plusieurs catégories immobilières ;
* être enrichi sous validation humaine.
