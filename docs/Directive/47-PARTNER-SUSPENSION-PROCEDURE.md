# Procédure de suspension des partenaires

Version 1.0

---

# 1. Objectif

Le présent document définit la suspension temporaire ou définitive d'un partenaire, d'une agence, d'un agent ou d'un prestataire.

---

# 2. Périmètre

Cette procédure couvre :

* les motifs de suspension ;
* la suspension préventive ;
* la suspension temporaire ;
* la suspension définitive ;
* la réactivation ;
* le recours ;
* la communication.

---

# 3. Acteurs concernés

* sécurité ;
* conformité ;
* opérations ;
* partenariat ;
* direction ;
* support.

---

# 4. Motifs possibles

* fraude ;
* plaintes répétées ;
* documents invalides ;
* comportement abusif ;
* non-respect des règles ;
* informations mensongères ;
* activité suspecte ;
* litiges non résolus.

---

# 5. Types de suspension

* préventive : protection immédiate ;
* temporaire : remise en conformité attendue ;
* définitive : rupture du partenariat ;
* partielle : restriction d'usage sur certains périmètres.

---

# 6. Étapes officielles

1. Déclenchement.
2. Analyse.
3. Proposition de décision.
4. Validation humaine.
5. Notification.
6. Application de la suspension.
7. Suivi.
8. Recours éventuel.
9. Réactivation ou clôture.
10. Archivage.
11. Reporting.

---

# 7. Autorité de décision

La décision revient selon le cas :

* à la sécurité pour la fraude ;
* à la conformité pour les écarts documentaires ;
* à l'opérationnel pour les manquements de suivi ;
* à la direction pour les cas sensibles.

---

# 8. Impact opérationnel

La suspension peut entraîner :

* la mise hors ligne de nouveaux biens ;
* la mise en revue de biens publiés ;
* la limitation de certaines conversations ;
* la suspension de certaines mises en relation ;
* le gel de certains suivis commerciaux ;
* le traitement spécifique des paiements déjà engagés selon les règles Campay.

---

# 9. Réactivation

La réactivation n'est possible que si :

* le motif de suspension a disparu ;
* les documents sont à jour ;
* la décision de reprise est validée par un humain ;
* l'historique reste conservé.

---

# 10. Notifications

Notifier :

* le partenaire ;
* l'agence ;
* le responsable interne ;
* la sécurité si nécessaire ;
* la finance si impact paiement ;
* le support pour suivi.

---

# 11. Reporting

Suivre au minimum :

* nombre de suspensions ;
* motifs de suspension ;
* durée moyenne de suspension ;
* nombre de réactivations ;
* nombre de suspensions définitives ;
* cas liés aux fraudes ;
* cas liés aux litiges.

---

# 12. Règles absolues

* ne jamais suspendre sans motif documenté ;
* ne jamais cacher une suspension ;
* ne jamais supprimer l'historique ;
* ne jamais créer de commission immobilière ;
* ne jamais réactiver sans validation ;
* ne jamais contourner la sécurité.

---

# 13. Objectif final

Cette procédure protège LAWIM en permettant une suspension claire, proportionnée, tracée et réversible lorsque cela est justifié.
