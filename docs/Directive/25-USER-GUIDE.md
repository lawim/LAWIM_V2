# 25-USER-GUIDE.md

# Guide utilisateur officiel

Version 1.0

---

# CHAPITRE 1 — OBJECTIF

Le présent document explique l'usage de LAWIM par les différents profils utilisateurs.

Il constitue la référence unique pour :

* le demandeur ;
* le propriétaire ;
* le détenteur ;
* l'agence ;
* l'agent ;
* le partenaire ;
* l'équipe LAWIM ;
* l'administrateur ;
* le paiement Campay.

---

# CHAPITRE 2 — GUIDE DEMANDEUR

Le demandeur peut :

* exprimer son besoin ;
* corriger ses critères ;
* consulter les propositions ;
* accepter ou refuser un bien ;
* suivre les visites et la mise en relation ;
* recevoir des notifications.

Le demandeur ne doit pas payer une commission sur une vente ou une location immobilière.

---

# CHAPITRE 3 — GUIDE PROPRIÉTAIRE

Le propriétaire peut :

* publier ou faire publier un bien ;
* compléter les informations ;
* valider les propositions ;
* suivre les visites ;
* demander les services LAWIM utiles.

Le propriétaire reste libre de ses décisions dans le respect des règles de LAWIM.

---

# CHAPITRE 4 — GUIDE DÉTENTEUR

Le détenteur peut :

* gérer un bien au nom d'un tiers si son mandat le permet ;
* valider les échanges autorisés ;
* suivre les services liés au bien ;
* consulter l'historique utile.

Le détenteur ne peut agir que dans le périmètre autorisé.

---

# CHAPITRE 5 — GUIDE AGENCE ET AGENT

L'agence et l'agent peuvent :

* gérer un portefeuille ;
* qualifier des besoins ;
* publier des biens ;
* suivre les leads ;
* organiser les visites ;
* accompagner les services.

Chaque action doit respecter les droits et le mandat reçus.

---

# CHAPITRE 6 — GUIDE PARTENAIRE, ÉQUIPE LAWIM ET ADMINISTRATEUR

Le partenaire peut suivre les services qui lui sont confiés.

L'équipe LAWIM peut :

* superviser ;
* contrôler ;
* aider ;
* administrer ;
* résoudre les incidents.

L'administrateur peut gérer les validations, les paramètres et la supervision des paiements Campay.

---

# CHAPITRE 7 — PARCOURS DE PAIEMENT CAMPAY

Pour les services payants :

1. le service est proposé ;
2. le paiement est initié ;
3. la confirmation backend est attendue ;
4. le service est activé après confirmation ;
5. un reçu ou un statut est affiché.

Le paiement ne déclenche jamais une commission immobilière.

---

# CHAPITRE 8 — NOTIFICATIONS ET ASSISTANCE

Les notifications peuvent indiquer :

* un matching ;
* une visite ;
* une relance ;
* un paiement confirmé ;
* un échec ;
* une expiration ;
* une action requise.

L'assistance LAWIM doit rester accessible pour les cas complexes.

---

# CHAPITRE 9 — BONNES PRATIQUES

Les utilisateurs doivent :

* fournir des informations exactes ;
* corriger rapidement les erreurs ;
* respecter les rendez-vous ;
* suivre les instructions officielles ;
* conserver les preuves utiles.

---

# CHAPITRE 10 — SUPPORT MULTILINGUE

L'utilisateur doit pouvoir utiliser LAWIM dans la langue de son choix lorsque la langue est disponible.

Les règles officielles sont les suivantes :

* la préférence de langue doit pouvoir être enregistrée ;
* l'interface doit suivre la langue active du profil ou de la session ;
* les aides, messages et confirmations doivent respecter la langue choisie ;
* le Français reste la langue par défaut ;
* en cas d'indisponibilité, LAWIM doit appliquer le fallback défini dans le référentiel de langue.

Le guide utilisateur s'appuie sur 30-I18N-L10N-REFERENCE.md, 30B-TRANSLATION-REFERENCE.md et 30C-LANGUAGE-DETECTION-REFERENCE.md.

---

# CHAPITRE 11 — TRACKING MARKETING POUR LES UTILISATEURS

Lorsqu'un utilisateur partage une publication ou une campagne, le code de tracking associé doit rester stable.

Le guide utilisateur doit aider à comprendre :

* d'où provient une publication ;
* comment un lead est attribué ;
* comment une conversion peut être reliée à une campagne ;
* comment une campagne peut être partagée sans perdre la traçabilité ;
* comment les statistiques personnelles ou d'équipe sont calculées.

Les utilisateurs ne doivent jamais manipuler les codes de tracking pour contourner les règles officielles.

---

# CHAPITRE 12 — OBJECTIF FINAL

Le guide utilisateur doit permettre à chaque profil de comprendre rapidement comment utiliser LAWIM de manière sûre, fluide et conforme aux règles officielles.
