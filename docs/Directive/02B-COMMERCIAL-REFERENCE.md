# Référentiel spécialisé des biens commerciaux

Version 1.0

---

# CHAPITRE 1 — OBJECTIF

Le présent document définit les spécificités de la famille commerciale.

Il hérite du référentiel maître des biens immobiliers.

---

# CHAPITRE 2 — PÉRIMÈTRE

Les biens commerciaux couvrent notamment :

* boutique ;
* bureau ;
* local commercial ;
* showroom ;
* restaurant ;
* espace de service ;
* cellule commerciale ;
* surface de vente ;
* plateau commercial.

---

# CHAPITRE 3 — ATTRIBUTS COMMERCIAUX SPÉCIFIQUES

Les attributs les plus utiles sont notamment :

* frontage ;
* visibilité ;
* accessibilité ;
* passage client ;
* parking client ;
* enseigne / signalétique ;
* surface utile ;
* sécurité ;
* puissance électrique ;
* eau ;
* ventilation ;
* conformité d'usage.

---

# CHAPITRE 4 — RÈGLES DE PUBLICATION

Un bien commercial doit indiquer clairement :

* l'usage autorisé ;
* l'emplacement ;
* la disponibilité ;
* la surface exploitable ;
* la compatibilité avec l'activité visée.

Le moteur ne doit pas confondre un usage commercial avec un usage résidentiel.

---

# CHAPITRE 5 — RÈGLES DE MATCHING

Le matching commercial valorise :

* la visibilité ;
* le flux ;
* la proximité de clientèle ;
* l'accès véhicule ou piéton ;
* les contraintes d'activité ;
* les charges d'exploitation.

---

# CHAPITRE 6 — RÈGLES DE VALIDATION

Un bien commercial est invalide si :

* l'usage annoncé n'est pas compatible avec les contraintes du site ;
* la visibilité ou l'accès est manifestement trompeur ;
* l'activité prévue est interdite ;
* le prix n'est pas cohérent avec le marché local.

---

# CHAPITRE 7 — TESTS SPÉCIFIQUES

Le référentiel commercial doit couvrir au minimum :

* boutique de rue ;
* bureau ;
* showroom ;
* local commercial de centre-ville ;
* restaurant / espace de restauration ;
* plateau commercial.

---

# CHAPITRE 8 — OBJECTIF FINAL

Le référentiel commercial doit permettre à LAWIM de qualifier et de matcher les espaces d'activité de manière exploitable pour les professionnels et les investisseurs.

---

# FIN DU DOCUMENT

Le présent **02B-COMMERCIAL-REFERENCE.md** constitue le référentiel spécialisé des biens commerciaux de LAWIM.
