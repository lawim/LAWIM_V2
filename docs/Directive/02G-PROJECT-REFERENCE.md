# Référentiel spécialisé des projets immobiliers

Version 1.0

---

# CHAPITRE 1 — OBJECTIF

Le présent document définit les spécificités de la famille des projets immobiliers.

Il hérite du référentiel maître des biens immobiliers.

---

# CHAPITRE 2 — PÉRIMÈTRE

Les projets immobiliers couvrent notamment :

* projet résidentiel ;
* projet commercial ;
* projet mixte ;
* lotissement ;
* programme neuf ;
* construction en cours ;
* opération de promotion ;
* pré-commercialisation.

---

# CHAPITRE 3 — ATTRIBUTS DE PROJET SPÉCIFIQUES

Les attributs les plus utiles sont notamment :

* promoteur ;
* phase du projet ;
* permis / autorisations ;
* nombre d'unités prévues ;
* date de livraison ;
* plan de paiement ;
* état d'avancement ;
* risque de livraison ;
* documents de projet ;
* rentabilité estimée ;
* calendrier ;
* niveau de commercialisation.

---

# CHAPITRE 4 — RÈGLES DE PUBLICATION

Un projet immobilier doit préciser :

* son promoteur ;
* sa phase ;
* son calendrier ;
* ses unités prévues ;
* son cadre juridique ;
* ses modalités de paiement lorsque l'offre est commercialisée.

---

# CHAPITRE 5 — RÈGLES DE MATCHING

Le matching projet valorise :

* le stade d'avancement ;
* le sérieux documentaire ;
* le calendrier de livraison ;
* la localisation ;
* le potentiel de rendement ;
* la cohérence du plan de paiement.

---

# CHAPITRE 6 — RÈGLES DE VALIDATION

Un projet immobilier est invalide si :

* le promoteur n'est pas identifiable ;
* la phase du projet est incompréhensible ;
* les autorisations critiques sont absentes ;
* le calendrier est incohérent ;
* les unités prévues ne peuvent pas être interprétées.

---

# CHAPITRE 7 — TESTS SPÉCIFIQUES

Le référentiel projet doit couvrir au minimum :

* programme neuf ;
* lotissement ;
* projet résidentiel ;
* projet commercial ;
* projet mixte ;
* construction en cours.

---

# CHAPITRE 8 — OBJECTIF FINAL

Le référentiel projet doit permettre à LAWIM de traiter les biens en développement, la pré-commercialisation et les opérations de promotion avec une logique claire et traçable.

---

# FIN DU DOCUMENT

Le présent **02G-PROJECT-REFERENCE.md** constitue le référentiel spécialisé des projets immobiliers de LAWIM.
