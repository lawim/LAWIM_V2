# CHAPITRE 1 — OBJECTIF

Le présent document définit les spécificités de la famille résidentielle.

Il hérite du référentiel maître des biens immobiliers et ne redéfinit pas les règles générales déjà portées par :

# CHAPITRE 2 — PÉRIMÈTRE

La famille résidentielle couvre notamment :

* chambre ;
* chambre moderne ;
* studio ;
* studio meublé ;
* appartement ;
* appartement meublé ;
* duplex ;
* maison ;
* villa ;
* immeuble résidentiel ;
* mini-cité ;
* résidence.

---

# CHAPITRE 3 — PRINCIPES SPÉCIFIQUES

Le résidentiel LAWIM doit respecter les usages du marché camerounais :

* langage simple et naturel ;
* reconnaissance des expressions locales ;
* qualification progressive ;
* priorité au confort réel plutôt qu'au jargon technique ;
* prise en compte de la vie familiale et de l'occupation réelle.

---

# CHAPITRE 4 — HÉBERGEMENT INDIVIDUEL

Les biens d'hébergement individuel sont :

* chambre ;
* chambre moderne ;
* studio ;
* studio meublé.

Spécificités :

* le nombre de chambres ne doit pas être demandé pour une chambre ou un studio ;
* la douche et la cuisine sont des critères déterminants ;
* le meublé est un critère clé pour le studio meublé ;
* l'accès indépendant et la qualité des finitions sont fortement discriminants.

---

# CHAPITRE 5 — LOGEMENTS FAMILIAUX

Les biens familiaux sont :

* appartement ;
* appartement meublé ;
* duplex ;
* maison ;
* villa.

Spécificités :

* le nombre de chambres devient pertinent ;
* l'organisation des pièces est importante ;
* le parking, l'accès et la sécurité pèsent dans le matching ;
* le standing peut compléter l'analyse sans remplacer les critères critiques.

---

# CHAPITRE 6 — RÉSIDENTIEL COLLECTIF

Les biens collectifs sont :

* immeuble résidentiel ;
* mini-cité ;
* résidence.

Spécificités :

* le nombre d'unités est important ;
* l'occupation doit être connue ;
* les services communs influencent le score ;
* la rentabilité peut être prise en compte pour les profils investisseurs ;
* les règles de copropriété ou de gestion commune doivent être signalées lorsqu'elles existent.

---

# CHAPITRE 7 — ATTRIBUTS RÉSIDENTIELS SPÉCIFIQUES

Les attributs souvent utilisés en résidentiel sont notamment :

* type de douche ;
* type de cuisine ;
* meublé / non meublé ;
* accès indépendant ;
* nombre de chambres ;
* nombre de pièces ;
* niveau de confort ;
* parking ;
* sécurité ;
* proximité route ;
* forage ;
* climatisation ;
* internet ;
* occupation.

Les valeurs officielles sont normalisées dans le catalogue d'attributs.

---

# CHAPITRE 8 — RÈGLES DE MATCHING RÉSIDENTIEL

Le matching résidentiel doit :

* exclure les types incohérents ;
* éviter les questions interdites ;
* distinguer chambre et studio ;
* distinguer logement familial et logement collectif ;
* tenir compte de la mobilité urbaine camerounaise ;
* utiliser les repères locaux lorsque l'adresse formelle est incomplète.

Le moteur ne doit pas exiger une précision de type hôtelier pour une simple recherche résidentielle.

---

# CHAPITRE 9 — RÈGLES DE VALIDATION RÉSIDENTIELLE

Un bien résidentiel est invalide si :

* son type est incohérent avec sa description ;
* un attribut interdit est renseigné comme critère principal ;
* la localisation est incompatible avec le type annoncé ;
* les données critiques sont absentes ;
* le prix ou la disponibilité ne peut pas être compris par le moteur.

---

# CHAPITRE 10 — TESTS SPÉCIFIQUES

Le référentiel résidentiel doit couvrir au minimum :

* recherche d'une chambre ;
* recherche d'une chambre moderne ;
* recherche d'un studio ;
* recherche d'un studio meublé ;
* recherche d'un appartement ;
* recherche d'un duplex ;
* recherche d'une villa ;
* recherche d'un immeuble résidentiel ;
* recherche d'une mini-cité.

Les tests doivent vérifier qu'aucune question inutile n'est posée.

---

# CHAPITRE 11 — OBJECTIF FINAL

Le référentiel résidentiel doit permettre à LAWIM de qualifier, matcher et publier les logements résidentiels de manière précise, naturelle et cohérente avec les usages du marché camerounais.
