# Référentiel spécialisé des biens agricoles

Version 1.0

---

# CHAPITRE 1 — OBJECTIF

Le présent document définit les spécificités de la famille agricole.

Il hérite du référentiel maître des biens immobiliers.

---

# CHAPITRE 2 — PÉRIMÈTRE

Les biens agricoles couvrent notamment :

* ferme ;
* plantation ;
* verger ;
* ranch ;
* serre ;
* parcelle agricole ;
* exploitation rurale ;
* domaine agricole.

---

# CHAPITRE 3 — ATTRIBUTS AGRICOLES SPÉCIFIQUES

Les attributs les plus utiles sont notamment :

* superficie exploitable ;
* qualité du sol ;
* source d'eau ;
* irrigation ;
* accès engins ;
* clôture ;
* relief ;
* culture ou activité ;
* tenure ;
* énergie disponible ;
* stockage ;
* habitation de service si applicable.

---

# CHAPITRE 4 — RÈGLES DE PUBLICATION

Un bien agricole doit préciser :

* la nature de l'exploitation ;
* la taille utile ;
* les ressources en eau ;
* l'accès ;
* les éventuelles infrastructures.

---

# CHAPITRE 5 — RÈGLES DE MATCHING

Le matching agricole valorise :

* la surface ;
* l'eau ;
* l'accès ;
* la qualité du sol ;
* la compatibilité avec la culture ou l'élevage visé ;
* la sécurité du site.

---

# CHAPITRE 6 — RÈGLES DE VALIDATION

Un bien agricole est invalide si :

* la vocation agricole est incohérente ;
* la surface est absente alors qu'elle est indispensable ;
* l'eau ou l'accès sont trompeurs ;
* le statut d'occupation ne peut pas être compris.

---

# CHAPITRE 7 — TESTS SPÉCIFIQUES

Le référentiel agricole doit couvrir au minimum :

* ferme ;
* plantation ;
* verger ;
* ranch ;
* serre.

---

# CHAPITRE 8 — OBJECTIF FINAL

Le référentiel agricole doit permettre à LAWIM de traiter les actifs agricoles avec des critères adaptés à l'exploitation rurale et aux réalités du terrain.

---

# FIN DU DOCUMENT

Le présent **02E-AGRICULTURAL-REFERENCE.md** constitue le référentiel spécialisé des biens agricoles de LAWIM.
