# Référentiel spécialisé des biens hôteliers

Version 1.0

---

# CHAPITRE 1 — OBJECTIF

Le présent document définit les spécificités de la famille hôtelière.

Il hérite du référentiel maître des biens immobiliers.

---

# CHAPITRE 2 — PÉRIMÈTRE

Les biens hôteliers couvrent notamment :

* hôtel ;
* guesthouse ;
* lodge ;
* aparthotel ;
* résidence hôtelière ;
* appartement de service ;
* hébergement meublé de courte durée.

---

# CHAPITRE 3 — ATTRIBUTS HÔTELIERS SPÉCIFIQUES

Les attributs les plus utiles sont notamment :

* nombre de chambres ;
* nombre de suites ;
* taux d'occupation ;
* réception ;
* restauration ;
* parking ;
* ménage ;
* sécurité ;
* classement / catégorie ;
* licences ;
* services inclus ;
* politique de séjour.

---

# CHAPITRE 4 — RÈGLES DE PUBLICATION

Un bien hôtelier doit préciser :

* sa capacité ;
* son niveau de services ;
* sa durée minimale de séjour si applicable ;
* ses équipements ;
* sa conformité d'exploitation.

---

# CHAPITRE 5 — RÈGLES DE MATCHING

Le matching hôtelier valorise :

* la capacité d'accueil ;
* la qualité des services ;
* la localisation ;
* l'état général ;
* le niveau de confort ;
* la disponibilité immédiate ou à date.

---

# CHAPITRE 6 — RÈGLES DE VALIDATION

Un bien hôtelier est invalide si :

* la capacité est incohérente ;
* les services annoncés ne sont pas plausibles ;
* les documents ou licences indispensables manquent ;
* la durée d'occupation n'est pas compréhensible.

---

# CHAPITRE 7 — TESTS SPÉCIFIQUES

Le référentiel hôtelier doit couvrir au minimum :

* hôtel ;
* guesthouse ;
* lodge ;
* aparthotel ;
* résidence hôtelière.

---

# CHAPITRE 8 — OBJECTIF FINAL

Le référentiel hôtelier doit permettre à LAWIM de qualifier des actifs d'hébergement professionnel avec des critères adaptés au séjour et à l'exploitation.

---

# FIN DU DOCUMENT

Le présent **02F-HOTEL-REFERENCE.md** constitue le référentiel spécialisé des biens hôteliers de LAWIM.
