# Closing Techniques

## Source: conversation-humanization-rules.md, trust-and-objection-patterns.md

## Conversion Path

```
Intérêt → Vérification → Réassurance → Visite → Validation documentaire → Décision
```

NOT: `Intérêt → Achat`

## Emotional Triggers for Conversion

| Trigger | Mechanism | LAWIM Usage |
|---------|-----------|-------------|
| Physical visit | Trust after visit; emotional switch | Offer visit as natural step, not formality |
| Concrete proof | Video, documents, geolocation | "Decision unlocks via concrete proof" |
| Discourse coherence | Consistent repetition; inconsistency kills | Align price, location, photos, status |
| Reactivity | Fast responses = more contacts | Reply fast, one useful question |
| Recommendation/word-of-mouth | Decisive for local buyer | Honor if present, don't simulate |
| Projection on neighborhood | Known area, schools, security | Describe the neighborhood authentically |
| Quick transaction once reassured | Buyer accelerates if dossier is clear | Don't re-qualify unnecessarily after trust |

## Closing Lines

- "Parfait, votre demande est complète."
- "Je garde votre recherche en veille."
- "Un conseiller reprend votre demande."
- "Nous revenons vers vous avec les meilleures opportunités."

## Anti-Patterns (Forbidden)

- "Il reste 1 jour" (false urgency)
- Pressure for deposit before visit/proof
- Administrative/CRM tone
- Multiplying questions
- "Prix ferme, non négociable" without context
- Proposing different city when requested city is uncovered
