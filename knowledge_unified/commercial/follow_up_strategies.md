# Follow-Up Strategies

## Source: conversation-patterns.md, trust-and-objection-patterns.md

## Core Principles

1. **Silence ≠ abandonment**: A silent lead is not automatically lost. Don't classify as uninterested based on silence alone.
2. **Long decision cycle**: A buyer may disappear, return after days or weeks, ask the same questions again - maintain welcoming tone.
3. **Social validation is normal**: "Mon grand frère doit voir" - keep dossier active, don't pressure.
4. **Every pattern ends concretely**: Either a next question, visible status update, or handoff/follow-up task.

## Follow-Up Intervals

| Priority | Interval | Channel |
|----------|----------|---------|
| URGENT | Within hours | WhatsApp/Telegram |
| HIGH | Daily | WhatsApp |
| NORMAL | Every 3 days | WhatsApp/Telegram |
| LOW | Weekly | Telegram |

## Follow-Up Rules

- One useful question only, not spam
- Re-use known context without reproach
- Never start a new thread when the user returns
- Keep the business identity of the request visible
- After visit: accompany naturally, don't re-qualify everything

## Diaspora Follow-Up

- Structured written summary
- Clear timelines
- Phone + written synthesis
- Documents and videos as standard

## After Silence

1. Keep the context
2. Short re-engagement: re-state the known need
3. One question about current status
4. If no response: WAITLISTED, not cancelled
