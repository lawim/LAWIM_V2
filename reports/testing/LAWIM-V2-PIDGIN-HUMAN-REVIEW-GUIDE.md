# LAWIM_V2 — PIDGIN HUMAN REVIEW GUIDE

## Purpose
Guide for native/regular Cameroon Pidgin English speakers to evaluate LAWIM's PCM translations.

## Evaluators needed
Minimum 3 evaluators who are:
- Native or regular Cameroon Pidgin English speakers
- Familiar with real estate vocabulary
- Able to read and assess written PCM

## Materials
- LAWIM_V2 web interface set to PCM mode
- This evaluation scorecard
- List of key phrases to test

## Test scenarios (15 minimum)
1. Property search in PCM
2. Qualification wizard in PCM
3. Property details display in PCM
4. AI assistant conversation in PCM
5. Error messages in PCM
6. Consent request in PCM
7. Visit request in PCM
8. Handover message in PCM
9. System notification in PCM
10. Welcome email in PCM
11. WhatsApp message in PCM
12. Form validation in PCM
13. Price display in PCM
14. Location description in PCM
15. Confirmation dialog in PCM

## Scoring criteria
Each phrase is scored 1-5 on:
- **Comprehensibility**: Can a typical PCM speaker understand it?
- **Naturalness**: Does it sound like natural PCM?
- **Professionalism**: Is it appropriate for business/real estate?
- **Accuracy**: Does it convey the correct meaning?
- **Respect**: Is it respectful and not caricatural?

## Minimum threshold
- Average score ≥ 4.0 across all categories
- No single category below 3.0
- No offensive or misleading phrase
