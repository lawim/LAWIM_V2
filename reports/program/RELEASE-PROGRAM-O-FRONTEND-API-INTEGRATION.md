# Release Program O — Frontend API Integration

## Objectif

Finaliser la phase de clôture du programme O côté frontend en intégrant l’application web à un SDK d’API dual-mode, avec conservation du comportement mock et activation des appels réels lorsque l’environnement l’indique.

## Ce qui a été livré

- Intégration du SDK frontend avec fallback mock conservé.
- Ajout d’un modèle d’authentification côté frontend et de routes protégées.
- Raccordement des pages web aux données backend attendues via l’API SDK.
- Vérifications frontend réalisées : typecheck, tests et build.

## Limites

- Aucun changement backend n’a été effectué.
- Les validations backend n’ont pas été relancées.

## Validation effectuée

- Frontend typecheck : OK
- Frontend tests : OK
- Frontend build : OK
