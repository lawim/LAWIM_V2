# Agent Review Checklist

## Pre-Modification

- [ ] Context read
- [ ] ADRs read
- [ ] Existing components searched
- [ ] Contracts identified
- [ ] Initial tests executed

## Implementation Safeguards

- [ ] Feature flags verified
- [ ] Shadow mode preserved
- [ ] No channel dependency
- [ ] No forbidden LLM dependency
- [ ] No duplicated business logic
- [ ] Idempotency verified
- [ ] Events verified
- [ ] Persistence verified
- [ ] Metrics verified

## Post-Modification

- [ ] Tests added
- [ ] Non-regression executed
- [ ] Validation level declared
- [ ] Evidence attached
- [ ] Worktree clean
- [ ] Commit identified
- [ ] Sync verified
