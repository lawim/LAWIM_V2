# Validation Process

## Etapes
1. Le ticket est defini par le Directeur General.
2. L'agent specialise prepare le livrable.
3. Le PMO enregistre l'avancement et les dependances.
4. Le QA verifie les criteres d'acceptation.
5. Le Directeur General accepte ou refuse.
6. Le PCC, les validations et les risques sont mis a jour.

## Validation Rules
- Aucun ticket incomplet ne peut etre valide.
- Aucun livrable hors perimetre ne peut etre accepte.
- Toute validation doit etre tracable.
- Toute reserve doit etre consigne dans le PCC ou les validations.
