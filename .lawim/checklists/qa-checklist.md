# QA Checklist

Utiliser cette checklist pour verifier la conformite fonctionnelle et la non-regression.

## Criteres d'acceptation
- [ ] Les criteres d'acceptation sont completes.
- [ ] Chaque critere est testable.
- [ ] Les cas limites sont identifies.
- [ ] Le comportement attendu est clair.

## Tests
- [ ] Les tests pertinents sont executes.
- [ ] Les epreuves de non-regression sont disponibles.
- [ ] Les resultats sont reproductibles.
- [ ] Les anomalies sont documentees.

## Evidence
- [ ] Les captures ou traces utiles sont jointes.
- [ ] Les ecarts sont qualifies.
- [ ] Les corrections necessaires sont listees.

## Cloture
- [ ] Le livrable est acceptable sans ambiguite majeure.
- [ ] Le statut de validation est trace.
