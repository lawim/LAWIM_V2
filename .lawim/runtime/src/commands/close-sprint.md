# Command: close-sprint

## Objective

Prepare and close a sprint in a traceable way.

## Inputs

- Sprint state.
- Ticket set.
- Validations.
- Archive pointers.

## Outputs

- Sprint closure bundle.
- Closure report seed.
- Final trace package.

## Critical errors

- Open ticket remains.
- Missing validation.
- Missing closure evidence.
- Conflicting sprint status.
