# Command: batch-run

## Objective

Execute a controlled batch of work items.

## Inputs

- Batch manifest.
- Ordered item list.
- Dependency map.
- Policy gates.

## Outputs

- Batch execution trace.
- Aggregate report seed.
- Follow-up planning input.

## Critical errors

- Invalid order.
- Duplicate item.
- Unresolved dependency.
- Inconsistent scope.
