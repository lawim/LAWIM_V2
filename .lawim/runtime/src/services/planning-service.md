# Service: Planning Service

## Objective

Prepare sequencing, sprint scope, and dependency order.

## Responsibilities

- Build execution order.
- Check scope boundaries.
- Prepare closure or batch input bundles.

## Inputs

- Ticket pool.
- Sprint context.
- Ordering rules.
- Constraint set.

## Outputs

- Plan.
- Schedule.
- Closure bundle input.

## Critical errors

- Missing dependency data.
- Invalid order.
- Scope overflow.
