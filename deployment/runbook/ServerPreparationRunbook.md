# Server Preparation Runbook

## Steps

1. Prepare OS packages and locale.
2. Create dedicated LAWIM users and groups.
3. Create application directories and ownership.
4. Configure firewall, fail2ban, and SSH access.
5. Configure Nginx, TLS, and service directories.
6. Validate server prerequisites before any deployment.

## Notes

All steps are documented as preparation-only actions for an eventual controlled migration.
